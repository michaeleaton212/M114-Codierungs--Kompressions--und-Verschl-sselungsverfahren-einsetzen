# Mithoel Foton

## 1 Kodierung
Umwandlung von Infos in eine bestimmte Darstellung, damit sie gespeichert oder übertragen werden können.

- QR Code
- Morsezeichen
- ASCII

---

## 2 Kompression
Reduzierung der Datenmasse einer Info, indem redundante Daten entfernt oder vereinfacht werden.

- ZIP
- Mp3
- JPEG

---

## Verschlüsselung
Wandelung lesbarer Daten in unlesbave.

- HTTPS
- PIN
- EZE
